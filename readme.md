📥 Installation
🐧 Linux (x64)
Download the Vortex-linux-x64.tar.gz file from the releases. Open your terminal in the download folder and run the following commands to extract and install

bash
# 1. Extract the downloaded archive
tar xzf Vortex-linux-x64.tar.gz
# 2. Enter the extracted directory
cd vortex_package
# 3. Run the installer with administrator privileges
sudo ./install.sh

Vortex will now be installed and accessible from your application menu!

📱 Android
Download the appropriate .apk file for your device architecture from the assets below (e.g., app-armeabi-v7a-release.apk or app-arm64-v8a-release.apk).
Open the downloaded file on your device.
If prompted, allow installation from unknown sources.
Install and enjoy!
